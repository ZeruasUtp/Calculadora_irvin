Manual

Para colocar una variable se coloca un asterisco:

Ejemplo:

3*x

Para colocar un exponente se colocan dos asteriscos:

Ejemplo:

3**2

Para colocar varias variables solas:

Ejemplo:

xy

Para colocar una variable y un exponente:

Ejemplo:

3*x**2

Para colocar varias variables:

Ejemplo:

3*x*y

Para colocar varias variables y exponentes:

Ejemplo:

3*x**2*y**2
